## Description

- Crud Departments (Cascade )

---

#### Technologies

- MYSQl or h2
- Bootstrap

1. Change in application.properties the Username && Thepass
2. Start Mysql
3. run <br>
   `.\mvnw.cmd spring-boot:run`
